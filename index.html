As I was finding and reading essays to prepare for assignment 2, I found the essay of Bubeck et al. (2023) more and more interesting. They published an article entitled "Sparks of Artificial Intelligence" Later, Bubeck also went to MIT and gave a presentation about this essay (2023). Bubeck made an interesting point in his speech. Usually when testing a machine learning model, people use some machine learning test set. For example, 1,000 multiple-choice questions on general knowledge of science can be counted as the test set. People then feed these questions to the model and measure the accuracy of the model's answers. But Bubeck said, "We're actually moving from machine learning to psychology... I don't want to use the machine learning test set. My assumption is that GPT-4 has seen any text on the Internet, including those tests... What we have to do is be creative and ask it some questions that have never appeared on the Internet."

Bubeck's words made perfect sense for me. When we can not understand something thoroughly, the first thing we can do is to experiment with it. One of the most interesting example that Bubeck gave in his presentation, is to ask GPT-4 to generate SVG code of a unicorn. SVG means "Scalable Vector Graphics". It is a text format to define some 2d shape or image with coordinates. For example, rect x="50" y="160" width="10" height="20" fill="white" stroke="black" stroke-width="2" the code like this will draw a rectange with the given xy coordinates, size, and colors on the screen. Just like rectangles, you can also draw segments, polygons, circles, and so on in svg. When I imitated Bubeck and asked ChatGPT to "draw" a unicorn, here is its result.
