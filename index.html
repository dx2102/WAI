I concluded that, large language models like GPT-4 can already do such a wide range of tasks, that it's not a problem to say that their intelligence is general. As I was finding and reading essays to prepare for assignment 2, I found the essay of Bubeck et al. (2023) more and more interesting. They published an article entitled "Sparks of Artificial Intelligence" Later, Bubeck also went to MIT and gave a presentation about this essay (2023). Bubeck made an interesting point in his speech. Usually when testing a machine learning model, people use some machine learning test set. For example, 1,000 multiple-choice questions on general knowledge of science can be counted as the test set. People then feed these questions to the model and measure the accuracy of the model's answers. But Bubeck said, "We're actually moving from machine learning to psychology... I don't want to use the machine learning test set. My assumption is that GPT-4 has seen any text on the Internet, including those tests... What we have to do is be creative and ask it some questions that have never appeared on the Internet."

Bubeck's words made perfect sense for me. When we can not understand something thoroughly, the first thing we can do is to experiment with it. Even if the conclusion may be misleading sometimes. One of the most interesting example that Bubeck gave in his presentation, is to ask GPT-4 to generate ".svg" code of a unicorn. "SVG" means "Scalable Vector Graphics". It is a text format to define some 2d shape or image with coordinates. For example, <rect x="50" y="110" width="100" height="50"> the code like this will draw a rectange with the given xy coordinates, size, and colors on the screen. Just like rectangles, you can also draw segments, polygons, circles, and so on in svg. When I imitated Bubeck and asked ChatGPT to "draw" a unicorn, here is its result. We can turn svg code into a real image by putting them into a website or put them into an editor called CodePen. (I am just using ChatGPT but Bubeck is using GPT-4, so the result may be different. Also, I did try a few time to get the best result.)

According to Bubeck (2023), the ability of ChatGPT to write svg code shows a few things. First, ChatGPT is good at using other tools based on text, and in psychology, the ability of humans to use tools is important. Besides using svg to draw, we can also tell ChatGPT to write codes, and actually run those codes. Bubeck also showed a few other examples in the presentation. For example, if we give ChatGPT a calculator, it will do a lot better in calculation. Also, if we connect ChatGPT to other text-to-image AI drawer, ChatGPT will be able to provide detailed descriptions of the image so that the AI drawer can draw better.

Secondly, according to Bubeck (2023), this shows that ChatGPT has some basic sense of space and world. ChatGPT is a language model, and it has only seen text and codes. It has never seen real images before. However, it can still draw a unicorn with most of the parts in the right place. Bubeck also showed example of ChatGPT "drawing" a map of a game based on the description it was given. For Bubeck, the point of this question is that, drawing a unicorn with SVG is a unique question that has never been asked on the Internet. Therefore it will require the real ability of ChatGPT. Therefore, ChatGPT will not be This means that some ideas of space hidden inside the text are also understood by ChatGPT.
